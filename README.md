# Fade Lab Website

Marketing website for Fade Lab Barbershop, built with Vite.

## Tech Stack

- Vite
- HTML/CSS/JavaScript

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Vercel Deployment

This project is configured for Vercel with:

- Build command: `npm run build`
- Output directory: `dist`
- Framework preset: `vite`

To deploy:

1. Push this project to GitHub.
2. Import the repository into Vercel.
3. Deploy (default settings will work with `vercel.json`).
